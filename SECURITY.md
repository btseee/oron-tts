# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

If you have discovered a security vulnerability in OronTTS, please send an email to our security team.
All security vulnerability reports will be managed with the utmost urgency.
